INTRODUCTION

Diba carousel module creates a block carousel based on Bootstrap carousel
functionality. The block is created programatically and is ready to use in 5
minutes, doesn't need views or configure extra modules.

 * For a full description of the module, visit the project page:
   https://drupal.org/project/diba_carousel

 * To submit bug reports and feature suggestions, or to track changes:
   https://drupal.org/project/issues/diba_carousel


REQUIREMENTS
------------

No special requirements.


INSTALLATION
------------

1. Enable the module.
2. Go to: Administration > Estructure > Block layout.
3. Place a Diba carousel block in a region.
4. Configure the block settings.


CONFIGURATION
-------------

Configure carousel placed carousels in block settings.
